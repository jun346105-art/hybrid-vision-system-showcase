# Hybrid Vision System Project Showcase

这个目录是用于网投简历的静态项目展示页。

## 本地查看

直接打开 `index.html`，或在本目录启动一个静态服务：

```powershell
python -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 部署

可将 `project_showcase/` 目录发布到 GitHub Pages、Gitee Pages 或任意静态站点托管服务。

## 素材来源

`assets/` 中图片来自当前项目的 `output/` 目录，页面文案基于 `main_pipeline.py`、`run_experiment.py`、`core/` 和 Phase 1/2/3 summary 文件。
